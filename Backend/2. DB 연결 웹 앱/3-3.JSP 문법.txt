<실습을 통해 JSP 문법 알아보기>

[스크립트 요소의 이해]

JSP페이지에서는
    선언문(Declaration) : <%! %> 전역변수 선언 및 메서드 선언에 이용
    스크립트릿(Scriptlet) : <% %> 프로그래밍 코드 기술에 사용
    표현식(Expression) : <%= %> 화면에 출력할 내용 기술에 사용
이라는 3가지의 스크립트 요소를 제공


[선언문(Declaration)]

- 선언문 : <%! %>
- 선언문은 JSP페이지 내에서 필요한 멤버변수나 메서드가 필요할 때 선언해 사용하는 요소
- 선언문의 문법
    <%! 문장 %>

- 선언문에서 변수 및 메서드 선언
    <%!
        String id = "u001"; //멤버변수 선언
        public String getId() {//메서드 선언
            return id;
        }
    %>


[스크립트릿(Scriptlet)]

- 스크립트릿 : <% %>
- 가장 일반적으로 많이 쓰이는 스크립트 요소
- 주로 프로그래밍의 로직을 기술할 때 사용
- 스크립트릿에서 선언된 변수는 지역변수(service메서드 안에 선언되는 변수)
- 스크립트릿의 문법
    <% 문장%>

ex)
    <%
        for(int i = 1; i <= 5; i++){
    %>
    <H<%=i %>> 아름다운 한글 </H<%=i %>>
    <%
        }
    %>


[표현식(Expression)]

- 표현식 : <%= %>
- JSP페이지에서 웹 브라우저에 출력할 부분을 표현
  즉, 화면에 출력하기 위한 것
- 스크립트릿 내에서 출력할 부분은
  내장객체인 out객체의 print()또는 println()메서드를 사용해서 출력
- 표현식의 문법
    <%=문장%>


[주석(Comment)]

- JSP페이지에서 사용할 수 있는 주석
    HTML주석, java주석, JSP주석

- HTML주석
    * HTML주석은 <!--로 시작해서 -->로 끝나는 형태
    * JSP주석이 아니기 때문에 JSP가 java로 바뀔 때도 같이 변환됨
    * java가 servlet으로 실행될 때도 포함되어 있음
    * HTML주석은 HTML주석을 사용한 페이지를 웹에서 서비스할 때 화면에 주석 내용이 표시되지는 않으나,
      [소스보기]를 실행하면 HTML주석의 내용이 화면에 표시
    * HTML주석의 예시
    <!--HTML주석입니다.-->

- JSP주석
    * JSP페이지에서만 사용되며 <%--로 시작해서--%>로 끝나는 형태
    * JSP주석은 해당 페이지를,
        웹 브라우저를 통해 출력 결과로서 표시하거나,
        웹 브라우저 상에서 소스 보기를 해도 표시되지 않음.
        또한 JSP주석 내에 실행코드를 넣어도 그 코드는 실행되지 않음.
    * JSP주석의 예시
    <%--JSP주석입니다.--%>

- java주석
    * java주석은 //, /**/을 사용해서 작성
    * JSP주석이 아니기 때문에 JSP가 java로 바뀔 때까지는 바뀜.
    * java가 servlet으로 실행될 때는 실행되지 않음
    * //은 한 줄짜리 주석을 작성할 때 사용되고,
      /**/은 여러 줄의 주석을 작성할 때 사용
    * 스크립트릿이나 선언문에서 사용되는 주석으로, 자바와 주석 처리 방법이 같음
    * java주석의 예시
        //주석
        /*주석
        여러 줄에 걸친 주석이다.
        */
